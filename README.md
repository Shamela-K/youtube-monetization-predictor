# YouTube Content Monetization Predictor

## Project Overview
This project predicts YouTube advertisement revenue using machine learning techniques. 
The model analyzes various video performance metrics such as views, likes, comments, 
watch time, subscribers, and other contextual features to estimate the potential 
advertising revenue generated by a video.

The goal of this project is to understand which factors influence monetization and 
build a predictive model that can estimate ad revenue based on video performance data.

---

## Project Objectives
- Analyze factors affecting YouTube ad revenue
- Build a regression model to predict revenue
- Evaluate multiple machine learning algorithms
- Deploy an interactive Streamlit web application for predictions

---

## Dataset
The dataset contains information about YouTube video performance including:

- Views
- Likes
- Comments
- Watch Time
- Video Length
- Subscribers
- Category
- Device Type
- Country

Target Variable:
- Ad Revenue

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

---

## Machine Learning Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor
- Support Vector Regressor

The best performing model was selected based on evaluation metrics.

---

## Model Evaluation Metrics
The models were evaluated using:

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

These metrics help measure how accurately the model predicts revenue.

---

## Streamlit Web Application
An interactive Streamlit application was developed to allow users to:

- Input video performance metrics
- Predict advertisement revenue
- Visualize feature influence using charts

Users can experiment with different values and instantly see predicted revenue.
